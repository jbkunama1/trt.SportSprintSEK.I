# Materialliste

## Sprint (DS 1–4)

| Material | Menge | Verwendung |
|---|---|---|
| Hütchen / Markierungsteller | 20–30 | Bahnen, Depots, Stationen |
| Startblock | 1–2 (optional) | DS 2, DS 4 |
| Baustellenband / Flatterband | 2–3 Stück | Fallstart-Korrektur, Baustellenbandfang |
| Spielkarten (Skat/Jass) | 1–2 Spiele | Kartensprint |
| Tennisbälle / kleine Gegenstände | 5–10 | Frühstarter, Risikosprint |
| Stoppuhr / Smartphone-App | 4–6 | Zeitnahme |
| Staffelstab | 4–6 | DS 4 |
| Puzzleteile (laminiert) | 1–2 Sets | Puzzle-Sprint |

## Weitsprung (DS 1)

| Material | Menge | Verwendung |
|---|---|---|
| Springseile | 1 pro Kind | Erwärmung, Hindernisfangis |
| Dünne Turnmatten | 8–12 | Pfützen, Seerosenblätter, Einfrieren |
| Weichbodenmatte | 1–2 | Skispringer-Landung, Flieger |
| Reifen | 15–25 | 5-Sprung-Challenge |
| Schaumstoffblöcke | 6–10 | Immer weiter! |
| Spielkarten | 1 Spiel | Jasskartenstafette |
| Hütchen | 10–15 | Zonen-Weitsprung |
| Rechen | 1 | Sandgrube (Zinken nach unten!) |
